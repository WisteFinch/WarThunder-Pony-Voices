# 战争雷霆小马语音包 WarThunder Pony Voice Pack

使用小马语音传播友谊魔法，Yay！

本语音包包含海陆空战语音及电台语音

- 海（正在重新制作以适配新版本）、陆、空战语音支持所有国家

- 电台语音支持所有语种

所有音频皆使用 [15.ai](https://15.ai/) 合成
